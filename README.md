## Features
- Users :
    - view book catalog
    - borrow book transaction
    - search & filter book by its categories
- Admins :
    - analytics for book transaction
    - book transaction management
    - categories management
    - books management

<br/>

## Prerequisite
* Laravel 11
* PHP 8.2+
* MySQL 8.3+
* AWS S3 Configuration

<br/>

## How to build
1. Git clone this repo
   
   ```$ git clone https://github.com/ai-null/perpus_digital.git```
2. Run composer install to install dependencies
   
   ```$ composer install```
3. Run migrate
   
   ```$ php artisan migrate```
4. (Optional) run seeder
   
   ```$ php artisan db:seed```
5. Run the program

   ```$ php artisan serve```
